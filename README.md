# 📱 TripWise - Travel Planner Android App

TripWise is a native Android application that helps users plan trips, track expenses, and manage travel notes efficiently.

---

## 🚀 Features

* 🗺️ Plan and organize trips
* 💰 Track expenses for each trip
* 📝 Add notes and reminders
* 📊 Simple and user-friendly interface

---

## 🛠️ Tech Stack

* Java
* Android Studio
* SQLite Database

---

## ▶️ How to Run

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run the app on emulator or device
